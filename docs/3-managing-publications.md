# 3. Managing Publications

This section explains how to manage the list of publications and software packages.

## ðŸ“š Accessing Publications

1.  In the CMS sidebar, click on **"Publications"**.
2.  You will see two main sections (or tabs):
    *   **Publications List**: The list of papers and books.
    *   **Software Packages**: A list of software developed by the group.

## ðŸ“„ Adding a Publication

Click on **"Publications List"** and then **"Add item"** (or edit an existing one).

### Fields Explained
*   **Title**: The full title of the paper or book.
*   **Year**: The year of publication (e.g., "2024").
*   **MR Number**: The MathSciNet ID (optional).
*   **Authors**: List of authors (e.g., "Author One, Author Two").
*   **Journal Details**: Where it was published (e.g., "J. Number Theory, Vol 12, pp 1-10").
*   **Status**: Select the current status:
    *   `Preprint`
    *   `Journal Article`
    *   `Book`
    *   `Submitted`
    *   `Other`
*   **Type**: Select `Article`, `Book`, or `Other`.

### Detailed Info (Subpage)
*   **Publisher Details**: For books.
*   **ISBN**: For books.
*   **Appendix**: Extra information or appendices.
*   **Content / Abstract**: The abstract or summary of the paper.

### Links and PDFs
*   **Links**: Add external links (e.g., to arXiv).
    *   `Label`: e.g., "arXiv".
    *   `URL`: The link address.
*   **PDFs**: Upload the PDF of the paper.
    *   `Label`: e.g., "Preprint PDF".
    *   `File`: Upload the file here.

## ðŸ’» Managing Software

Click on **"Software Packages"** to add or edit software.

*   **Title**: Name of the software.
*   **Description**: What the software does.
*   **Author**: Who wrote it.
*   **Link URL**: Link to the code or website.
*   **Link Text**: e.g., "View on GitHub".
*   **Thesis URL**: If it's part of a thesis.
