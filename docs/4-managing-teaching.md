# 4. Managing Teaching

This section explains how to manage the Teaching page, including courses, seminars, and lecture notes.

The teaching section is organized hierarchically:
1.  **Year** (e.g., 2024)
2.  **Semester** (e.g., Winter, Summer)
3.  **Courses** (The actual classes)

## âž• Adding a New Course

1.  In the CMS sidebar, click on **"Teaching"**.
2.  You will see a list of **Courses by Year**.

### Step 1: Find or Create the Year
*   If the year already exists, click on it to expand it.
*   If not, click **"Add item"** to create a new year entry (e.g., "2025").

### Step 2: Find or Create the Semester
*   Inside the Year, look for **"Semesters"**.
*   Click **"Add item"** to add a semester.
*   **Term**: Select `Winter` or `Summer`.

### Step 3: Add the Course
*   Inside the Semester, look for **"Courses"**.
*   Click **"Add item"** to add a new course.

### Course Fields Explained
*   **Course Title**: The name of the lecture or seminar.
*   **Instructor**: Who is teaching it.
*   **Course Type**: Select `Lecture`, `Seminar`, `Proseminar`, `Exercise`, or `Other`.
*   **Description**: A short description shown on the main list.
*   **Course Details (Subpage Only)**: The full content for the course page. You can add tables, lists, and formatted text here.

### Links and Materials
*   **Links**: Add links to external resources (e.g., Moodle, LSF).
    *   `Label`: e.g., "Moodle".
    *   `URL`: The link.
*   **PDFs**: Upload lecture notes or exercise sheets.
    *   `Label`: e.g., "Lecture Notes 1".
    *   `File`: Upload the PDF file.
