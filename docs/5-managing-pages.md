# 5. Managing Static Pages

This section explains how to edit the content of the Home, Research, Links, and Contact pages.

## üè† Home Page

1.  In the CMS sidebar, click on **"Home Page"**.
2.  **Hero Section**:
    *   `Image`: The large background image at the top.
    *   `Title`: The main title (e.g., "AG Computational Arithmetic Geometry").
    *   `Subtitle`: The text below the title.
    *   `Description`: The introductory text.
3.  **Feature Cards**: These are the boxes highlighting key areas.
    *   `Title`: Card title (e.g., "Research").
    *   `Icon Class`: FontAwesome icon name (e.g., `fas fa-book`).
    *   `Content`: Text inside the card.
    *   `Link`: Where the card links to (e.g., `/research/`).

## üî¨ Research Page

1.  In the CMS sidebar, click on **"Research Page"**.
2.  **Research Areas**: You can add or edit specific research topics.
    *   `Title`: Name of the research area.
    *   `Icon Class`: Icon for the section.
    *   `Content`: Detailed description of the research.

## üîó Links Page

1.  In the CMS sidebar, click on **"Links Page"**.
2.  **Link Groups**: Links are organized into groups (e.g., "University", "Math Institutes").
    *   `Group Title`: Name of the group.
    *   `Icon Class`: Icon for the group.
    *   **Links**: Add individual links inside the group.
        *   `Title`: Text for the link.
        *   `URL`: The website address.

## üìû Contact Page

1.  In the CMS sidebar, click on **"Contact Page"**.
2.  **Address**: Update the physical address and map link.
3.  **Contact Details**: Update phone, fax, and general info.
4.  **Email Section**: Update the email address and surrounding text.

---

**Congratulations!** You now know how to manage the entire website.
