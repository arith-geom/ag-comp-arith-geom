# Website Management Documentation

Welcome to the documentation for managing the **AG Computational Arithmetic Geometry** website.

This guide is designed to help you update and maintain the website content without needing any programming knowledge. We use a system called **Pages CMS** which provides a user-friendly interface for editing text, uploading images, and managing lists like publications and members.
